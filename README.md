# LeanLift

A Vite + React prototype for a lightweight workout tracker.

## Run locally

```bash
npm install
npm run dev
```

## Current structure

- `src/App.jsx` — app wiring and state
- `src/components/` — UI pieces split into manageable files
- `src/lib/` — exercise data, storage helpers, analytics helpers

## Notes

This is still a prototype, but it is now split into repo-friendly files so you can keep building without fighting one giant canvas file.
